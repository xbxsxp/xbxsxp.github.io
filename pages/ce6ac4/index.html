山脉数组的峰顶索引

符合下列属性的数组 arr 称为 山脉数组 ：

  • 存在 其中 使得： 并且

示例1:

输入：arr = [0,1,0]
输出：1

2

示例2:

输入：arr = [0,2,1,0]
输出：1
1

示例3:

输入：arr = [0,10,5,2]
输出：1
1
2
class Solution:
    def peakIndexInMountainArray(self, arr: List[int]) -> int:
        n = len(arr)
        left = 0
        right = n - 2

        while left < right:
            mid = left + (right - left) // 2
            if arr[mid] < arr[mid + 1]:
                left = mid + 1
            else:
                right = mid

        return left
